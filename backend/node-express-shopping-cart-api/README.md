# shopping-cart-api

A shopping cart API using MongoDB, Express and NodeJs

#### Run locally on your machine

You need to have installed Node and NPM.

### Clone the repo

```bash
$ git clone https://github.com/abdulsamad245/fanny-assessment.git
```

### Install dependencies

```bash
$ npm install
```

### Start NodeJS server

```bash
$ npm start
```

### For Nodemon usage

```bash
$ nodemon app.js
```
